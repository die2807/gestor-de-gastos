# Gestor de gastos (MVP)

## Formato de datos (CSV)
Cada gasto se guarda con:
- ID (numero único)
- Fecha (YYYY-MM-DD)
- Categoría (Texto)
- Monto (número decimal)
- Descripción (texto)
- Necesidad (numero 1 - 5)
